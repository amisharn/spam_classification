# spam_classification
Live at: https://spamclassification-r6bqhzdvhsfck6qyd57hy9.streamlit.app/

I learnt different classification algorithms, found Support Vector Model the most effective for this dataset, exported the model, created a streamlit app, connected it with the FASTAPI backend and deployed using Render and Streamlit UI. 

I did this project in order to learn a full ml workflow from EDA to deployment. Will be improving this later.
